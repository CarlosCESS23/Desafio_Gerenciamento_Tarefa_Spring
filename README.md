# Desafio sistema de gerenciamento de tarefas

## Descrição:

É desenvolvido uma aplicação web utilizando uma linguagem de programação e um framework, onde a aplicação deve ser um sistema de gerenciamento de tarefas, onde o usuário podem criar, visuazliar, editar e excluir tarefas


## Requisito:

* Deve-se usar Banco de dados
* Campos mínimos da entidade de tarefa: **Nome**,**Descrição**,**Realizado**,**Prioridade**
* Criar Crud de Tarefa

## Tecnologias que são usados para esse desafio:

#### Linguagem de programação: JAVA
#### Framework : Spring
